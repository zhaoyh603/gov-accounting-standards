# 电子凭证会计数据标准（推广应用版）

> **财会〔2025〕4号**

为贯彻落实《中共中央关于进一步全面深化改革、推进中国式现代化的决定》关于深化财税体制改革的决策部署，推动电子凭证会计数据标准应用，促进全社会会计信息化建设，根据《中华人民共和国会计法》、《会计改革与发展"十四五"规划纲要》、《会计信息化发展规划（2021-2025年）》有关要求，财政部制定了电子凭证会计数据标准（推广应用版）。

---

## 一、电子凭证会计数据标准概述

### （一）什么是电子凭证会计数据标准

电子凭证会计数据标准是指对电子凭证的生成、传输、解析、入账、归档等全过程进行规范的技术标准。它包括数据格式标准、接口标准、安全标准等内容，旨在实现电子凭证在不同系统之间的互认互通。

### （二）电子凭证会计数据标准的意义

1. **提高会计工作效率**：实现电子凭证全流程自动化处理，减少人工录入，提高处理效率。
2. **降低会计工作成本**：减少纸质凭证的打印、传递、存储成本，实现无纸化办公。
3. **提升会计数据质量**：通过结构化数据自动解析，减少人工录入错误，提高数据准确性。
4. **促进数据共享共用**：实现会计数据跨部门、跨系统、跨层级的共享共用，提升数据价值。

### （三）电子凭证会计数据标准的适用范围

电子凭证会计数据标准适用于：
- 企业、行政事业单位等组织的会计核算
- 会计软件的开发和应用
- 电子凭证服务平台的设计和运营
- 财政、税务、审计等部门的会计监督

---

## 二、电子凭证会计数据标准的主要内容

### （一）数据格式标准

电子凭证会计数据标准规定了电子凭证的数据格式，包括：

1. **基础信息**：凭证编号、日期、摘要、金额等
2. **科目信息**：会计科目编码、名称、借贷方向、金额等
3. **辅助信息**：部门、项目、客户、供应商等辅助核算信息
4. **附件信息：电子发票、合同、审批单等附件的关联信息

数据格式采用 XML 或 JSON 等结构化数据格式，便于计算机解析和处理。

### （二）接口标准

电子凭证会计数据标准规定了电子凭证的接口标准，包括：

1. **数据输入接口**：用于接收外部系统传入的电子凭证数据
2. **数据输出接口**：用于向外部系统输出电子凭证数据
3. **数据查询接口**：用于查询电子凭证的状态和内容
4. **数据验证接口**：用于验证电子凭证的合法性和完整性

接口采用 RESTful API 或 Web Service 等标准化接口协议，便于系统集成。

### （三）安全标准

电子凭证会计数据标准规定了电子凭证的安全标准，包括：

1. **身份认证**：采用数字证书、生物识别等技术确认用户身份
2. **数据加密**：采用 SSL/TLS、国密算法等技术保护数据传输和存储安全
3. **数字签名**：采用电子签名技术确保电子凭证的完整性和不可否认性
4. **访问控制**：采用权限管理、审计日志等技术控制数据访问和操作

安全标准符合国家密码管理相关规定和网络安全等级保护要求。

## 三、电子凭证会计数据标准的应用流程

### （一）电子凭证生成

1. **原始凭证电子化**：将纸质原始凭证扫描或拍照转换为电子图像
2. **结构化数据提取**：通过 OCR 技术或手工录入提取凭证的关键信息
3. **电子凭证封装**：将凭证图像和结构化数据按照标准格式封装为电子凭证
4. **数字签名**：对电子凭证进行数字签名，确保其完整性和不可否认性

### （二）电子凭证传输

1. **安全通道建立**：通过 SSL/TLS 建立安全传输通道
2. **身份认证验证**：验证发送方和接收方的身份信息
3. **电子凭证发送**：将电子凭证通过标准接口传输到接收方系统
4. **传输确认反馈**：接收方确认收到电子凭证并返回回执

### （三）电子凭证解析

1. **格式验证**：验证电子凭证的数据格式是否符合标准
2. **签名验证**：验证数字签名的有效性和完整性
3. **数据解析**：解析电子凭证中的结构化数据，提取会计要素
4. **数据映射**：将解析的数据映射到会计系统的数据结构中

### （四）电子凭证入账

1. **凭证审核**：对解析后的数据进行审核，确保其合法性和准确性
2. **凭证生成**：根据解析的数据自动生成会计凭证
3. **凭证记账**：将生成的会计凭证登记到账簿中
4. **凭证归档**：将电子凭证和相关资料归档保存

### （五）电子凭证归档

1. **归档整理**：按照档案管理要求对电子凭证进行分类整理
2. **元数据提取**：提取电子凭证的元数据，建立索引
3. **长期保存**：将电子凭证存储到可靠的存储介质中，确保长期可读
4. **档案管理**：建立电子档案管理制度，确保档案的安全和可用

## 四、电子凭证会计数据标准的实施要求

### （一）组织保障

1. **建立工作机制**：成立电子凭证会计数据标准实施工作领导小组，明确职责分工
2. **制定实施方案**：结合单位实际情况，制定详细的实施方案和时间表
3. **保障资源投入**：确保人力、物力、财力等资源的投入，保障实施工作顺利进行
4. **加强监督考核**：建立监督考核机制，定期检查实施进度和效果

### （二）技术保障

1. **系统升级改造**：对现有会计信息系统进行升级改造，支持电子凭证会计数据标准
2. **接口开发对接**：开发标准接口，实现与外部系统的数据对接和交换
3. **安全防护措施**：加强系统安全防护，确保电子凭证数据的安全性和完整性
4. **数据备份恢复**：建立数据备份和恢复机制，确保数据的可靠性和可用性

### （三）人员保障

1. **加强培训学习**：组织相关人员参加电子凭证会计数据标准培训，提高认识和技能
2. **明确岗位职责**：明确相关人员的岗位职责和工作要求，确保责任到人
3. **建立激励机制**：建立激励机制，调动相关人员的积极性和主动性
4. **加强团队建设**：加强团队建设，提高团队协作能力和工作效率

### （四）制度保障

1. **完善管理制度**：建立健全电子凭证管理制度，明确管理要求和工作流程
2. **规范操作流程**：制定电子凭证操作流程，规范各环节的操作行为
3. **强化内部控制**：加强内部控制，建立有效的监督制约机制
4. **定期评估改进**：定期评估电子凭证管理工作，及时发现问题并改进
